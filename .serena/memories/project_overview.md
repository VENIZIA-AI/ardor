# ARDOR Project Overview

## Purpose
ARDOR is a UI Kit library built by VENIZIA AI. It provides React UI components with Tailwind CSS v4 integration, Radix UI primitives, and shadcn/ui components for building admin interfaces and web applications.

## Type
- Bun-based monorepo
- Published NPM packages (@venizia/ardor-admin)
- MIT licensed
- Repository: https://github.com/VENIZIA-AI/ardor

## Main Package
`@venizia/ardor-admin` - Admin UI Kit with React 19, Radix UI, and Tailwind CSS v4

## Key Features
- React 19 components with TypeScript
- Radix UI primitives for accessible components
- shadcn/ui integration for base components
- Adaptive components that work across mobile/desktop (Dialog↔Sheet, Popover↔Drawer)
- Tailwind CSS v4 styling with CSS variables
- Auto-generated barrel exports for clean imports
