# @venizia/ardor-admin

1. Install `@venizia/ardor-admin` as a dependency by running the following command:

```bash
npm install @venizia/ardor-admin
```

2. Import the default theme variables from `@venizia/ardor-admin` inside your main `index.css` CSS file:

```css
@import url('https://fonts.googleapis.com/css2?family=Inter:ital,opsz,wght@0,14..32,100..900;1,14..32,100..900&display=swap');
@import '@venizia/ardor-admin/styles/default.css';
```

3. Configure the source files of `@venizia/ardor-admin` in your CSS:

```css
@source '../node_modules/@venizia/ardor-admin';
```
